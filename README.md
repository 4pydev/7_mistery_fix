# Решатель квадратных уравнений

Модуль quadratic_equation решает квадратное уравнение по его коэффициентам a, b и с. Если уравнение не имеет действительных решений возвращает значения корней None.

# Как использовать

Модуль содержит единственную функцию get_roots(a, b, c), принимающую в качестве аргументов коэффициенты квадратного уравнения и возвращающую root1 и root2 - корни уравнения.

В случае нулевого дискриминанта root2 принимает значение None, а в случае отрицательного дискриминанта оба корня принимают значение None.

Пример использования:

```python
from quadratic_equation import get_roots

...
root1, root2 = get_roots(k1, k2, k3)
```

# Как запустить

Скрипт требует для своей работы установленного интерпретатора Python версии 3.5

Запуск на Linux:

```bash
python tests.py # может понадобиться вызов python3 вместо python, зависит от настроек операционной системы
```

Запуск на Windows происходит аналогично.
